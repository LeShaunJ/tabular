    struct Tabular::Token

    • Tabular::Token
    • Struct
    • Value
    • Object

    Overview

    Represents a parameter whose name and aliases may be suggested and amtched during tab completion.

    Defined in:

    tabular/token.cr

    Constant Summary

    NONE = Token.new(:none)

    Basically, Token? minus the baggage.
